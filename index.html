persistent_ha1 by richardliao

This is a persistent sha1 package

The built-in hashlib.sha1 in python is a wrapper around a C library, which does not support persistence. There are some pure sha1 implementations support persistence, like: http://force.vnsecurity.net/download/rd/sha.py or http://code.google.com/p/slowsha/ The problem is that they are too slow than hashlib.sha1 to be used in production environment. This package adapts from the C sha1 module from python 2.4.6, add two methods: dumps and loads. So, the performance is the same as python's one, and we can save and restore the status at anytime. The dumped string is small(108 bytes).

Install

python setup.py build sudo python setup.py install
